📥 Быстрая установка

1
Скачайте архив с GitHub
git clone https://github.com/Prodebug/GameDevStudio.git
или скачайте ZIP-архив с сайта.
2
Перейдите в папку проекта
cd GameDevStudio
3
Установите зависимости (опционально)
pip install -r requirements.txt
Если у вас Python 3.8+, большинство библиотек уже встроены.
4
Запустите приложение
python main.py
✅ Готово! Наслаждайтесь разработкой.
